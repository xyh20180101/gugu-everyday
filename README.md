# GuGu Everyday

**勤奋创作每一天——创作进度管理与展示平台**

管理你的创作项目进度，生成展示页分享给他人，接受催更监督。

## 功能概览

- **项目管理** — 创建项目类型（进度/工序），管理多个项目及其进度
- **甘特图** — 首页按月展示项目时间线，直观查看各项目的时间安排
- **展示页** — 每个用户拥有独立的展示页，可选择公开哪些项目
- **催更** — 开启后其他人可通过展示页向你催更

## 快速开始

### 注册

1. 访问登录页，点击「注册」
2. 填写邮箱和密码
3. 点击邮件中的验证链接完成激活
4. 激活后即可登录

### 创建项目类型

项目类型决定了进度的计量方式：

| 类型 | 说明 | 示例 |
|------|------|------|
| 进度 | 用数字或时间量化 | 100章、4:00:00、50秒 |
| 工序 | 阶段性里程碑 | 草图→线稿→上色→细化→效果 |

可选填写「总进度」和「进度单位」，用于在展示页上显示完成百分比。

### 创建项目

- **类型** — 选择已创建的项目类型
- **开始/完工日期** — 填写后会显示在首页甘特图上
- **是否公开** — 控制该项目是否出现在你的展示页上
- **总进度/进度单位/工序** — 自动从项目类型继承，可单独覆盖

### 记录进度

- **进度类型项目** — 手动填写当前进度值（如"第50章"、"2:30:00"）
- **工序类型项目** — 从预设的工序中选择当前阶段

### 归档

完成或不再更新的项目可以归档。归档后：
- 不再显示在默认列表和展示页中
- 无法修改项目信息
- 可在「查看已归档」中取消归档

## 个人设置

| 设置项 | 说明 |
|--------|------|
| 用户名 | 显示在展示页上 |
| 公告 | 显示在展示页顶部 |
| 展示页标题 | 展示页的页面标题 |
| 启用展示页 | 开启后其他人才能访问你的展示页 |
| 允许催更 | 开启后其他人可通过展示页催更（同一项目同一 IP 5 分钟内最多 3 次） |
| 项目名称掩码 | 输入 1-2 个字符，展示页上会逐字替换项目名称（如用"*"替代） |

## 展示页

启用后，其他人可通过你的专属链接访问展示页：

- 查看你公开的项目及进度
- 查看公告
- 催更（需开启允许催更）

展示页链接可在「个人设置」中查看和复制。

## 安全说明

- 密码使用 PBKDF2 + SHA256 加盐存储（10 万次迭代）
- 登录和注册接口使用 PoW（工作量证明）防止暴力攻击
- 登录接口有 IP 级频率限制（60 秒内超过 20 次则拉黑 IP 10 分钟）
- 注册接口有 IP 级频率限制（60 秒/次）
- 注册需要邮箱验证激活
- 除展示页和催更外，所有接口均需身份认证

## 技术栈

| 层 | 技术 |
|----|------|
| 后端 | ASP.NET Core 10 + ABP Module + EF Core + MySQL + Redis |
| 前端 | Vue 3 + Fluent UI Web Components + Naive UI + ECharts |
| 部署 | Docker (阿里云) + Cloudflare Pages (前端) |
| AI Coding | Mimo Code + mimo v2.5 pro |

### 核心实现

| 模块 | 技术方案 | 说明 |
|------|----------|------|
| 敏感词过滤 | Aho-Corasick + Trie 树 | 启动时构建自动机，O(n) 多模式匹配，支持敏感词库热加载。敏感词词库来自[Sensitive-lexicon](https://github.com/konsheng/Sensitive-lexicon) |
| 分布式 ID | Snowflake 改进算法 | 41位时间戳 + 12位序列号，单机每毫秒可生成 4096 个 ID |
| 密码安全 | PBKDF2 + SHA256 | 10万次迭代加盐哈希，使用 `FixedTimeEquals` 防时序攻击 |
| 防暴力破解 | PoW + IP 频率限制 | 登录/注册前客户端需完成 SHA256 计算挑战；登录接口 60 秒内同一 IP 超过 20 次自动拉黑 10 分钟 |
| 甘特图可视化 | ECharts 自定义渲染 | 支持按月时间线展示，自定义项目颜色与进度条样式 |
| 认证授权 | JWT + Redis 黑名单 | Token 存储在 Redis，支持主动失效；登出即失效 |
| 邮件验证 | MailKit + 分布式缓存 | 注册/重置密码邮件验证，Token 存 Redis 带过期时间 |
| 内容审核 | 自定义 Attribute + EF Core 拦截 | 通过 `[SensitiveLexicon]` 标记字段，SaveChanges 自动校验 |
| 国际化 | vue-i18n | 支持中/英双语，语言切换无需刷新 |
| 组件库 | Fluent UI Web Components | 微软设计体系，原生 Web Components，无框架依赖 |
